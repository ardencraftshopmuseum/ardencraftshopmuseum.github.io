WordPress Themes and Site Administration, Thurs., 5/21 2-3:30 pm

Training |

This class provides you with several different features available in WordPress. At the end of this class, you will be able to:

  • Search for themes by keywords
  • Understand how to customize sites using the Customizer, Theme Options, Widgets, Menus and Plugins
  • Explore advanced themes (e.g., Divi)
  • Export and import from one WordPress site to another

This class is held in Smith 010 and lasts 90 minutes.
